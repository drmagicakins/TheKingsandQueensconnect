# TheKingsandQueensconnect
A community web app where members can connect and grow

### STOP AT - Why this works (Title reading)

Router, Navbar, Pages and Credits components have 
been modified to include new features such as 
online user badges, birthday spotlights, and 
a smart marquee. The LeaderBoard and Wallet 
components have been updated for better styling 
and functionality. New admin and community 
components have also been added to enhance 
the application's capabilities.
